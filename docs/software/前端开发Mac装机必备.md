# 前端开发 Mac 装机必备

## 梯子后不能只能拉取

![alt text](443.png)

1. 终端运行

```shell
code ~/.zshrc
```

2. 最后添加（以下命令，通过代理软件ClashX，右键，复制终端命令获取）

```shell
export https_proxy=http://127.0.0.1:7890 http_proxy=http://127.0.0.1:7890 all_proxy=socks5://127.0.0.1:7890
```

3. 运行

```shell
source ~/.zshrc
```
