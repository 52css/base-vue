# useDebounce 延迟响应

## 使用场景

- 对应一些高频响应输入，例如input输入查看data-list，我们只用最后一次响应数据

## API

| 参数 | 说明     | 类型                    | 默认值 | 版本 |
| ---- | -------- | ----------------------- | ------ | ---- |
| value   | 默认值 | `T` | \-     |      |
| delay   | 延迟时间, 默认`ms` | `number` | `1000`     |      |
