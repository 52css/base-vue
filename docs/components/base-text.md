## API

| 参数    | 说明                           | 类型                   | 默认值   | 版本 |
| ------- | ------------------------------ | ---------------------- | -------- | ---- |
| align   | 对其                           | `'left'` \| `'center'` | `left` |      |
| maxline | 最多显示多少行，超出显示省略号 | `number`               | \-       |      |
| suffix  | 后缀，常用资源省略号           | `string`               | \-       |      |
